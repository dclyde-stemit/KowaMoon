Endpoint
{version}/embed/movie/{id}
Valid parameters:
- {version} required determines embed version - v2 or v3.

- {id} required - from imdb.com or themoviedb.org , imdb id must have tt prefix.

Customization Parameters:
- poster: Determines if the poster image is shown.

- autoPlay: Controls whether the media starts playing automatically.

- startAt: Starts the video at the specified time in seconds.

- color: Changes the player's color (hex code format).

Examples
https://vidsrc.cc/v2/embed/movie/533535

https://vidsrc.cc/v2/embed/movie/tt6263850

https://vidsrc.cc/v3/embed/movie/533535?autoPlay=false

Endpoint
v2/embed/anime/{id}/{episode}/{type}
Valid parameters:
- {id} required - from anilist.co , myanimelist.net , imdb.com or themoviedb.org.

- anilist id must have ani prefix

- imdb id must have imdb prefix

- tmdb id must have tmdb prefix

- {episode} required - the episode number.

- {type} required - sub or dub.

Customization Parameters:
- autoPlay: Controls whether the media starts playing automatically.

- autoSkipIntro: Determines skip intro or outro.

Examples
https://vidsrc.cc/v2/embed/anime/21/2/sub

You can load your own subtitles by two methods:

Single file:
https://vidsrc.cc/v2/embed/movie/{id}?sub.file={sub_file}&sub.label={sub_label}

- {sub_file} required - Url to .vtt sub

- {sub_label} optional

Multiple file:
https://vidsrc.cc/v2/embed/movie/{id}?sub.info={sub_json}

- {sub_json} required

- A json file with this format:

  [ { file: "https://domain.com/file1.vtt", label: "English", kind: "captions" }, { file: "https://domain.com/file2.vtt", label: "Japanese", kind: "captions" } ]
                        
* {sub_file} or {sub_json} must have Access-Control-Allow-Origin: * header

Listen to player events to track user interactions. Events are sent via postMessage to the parent window.

Available Events:
- play Triggered when video starts playing

- pause Triggered when video is paused

- time Triggered periodically every five seconds

- complete Triggered when video ends

Event Data Structure:
  {
    type: "PLAYER_EVENT",
    data: {
      event: "play" | "pause" | "time" | "complete",
      tmdbId: number,
      duration: number,
      currentTime: number,
      mediaType: "movie" | "tv",
      season?: number,
      episode?: number
    }
  }
Implementation Example:
  window.addEventListener('message', (event) => {
    if (event.origin !== 'https://vidsrc.cc') return;
                              
    if (event.data && event.data.type === 'PLAYER_EVENT') {
    const { event: eventType, currentTime, duration } = event.data.data;
    // Handle the event
    console.log(`Player ${eventType} at ${currentTime}s of ${duration}s`);
    }
  })
